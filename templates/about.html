About This Project

This Master's project focuses on developing an AI-powered document summarization system capable of handling various document formats. The system aims to provide accurate and coherent summaries along with informative visualizations.

Plan for the Past Three Weeks

The plan for the past three weeks focused on enhancing the core functionalities of the document summarization system:

  • Enhanced Document Upload and Preprocessing: Implementing robust document upload for Word, PDF, text, and images, and developing comprehensive preprocessing logic with OCR for images.
  • Advanced Summarization Integration: Refining the integration with the open-source llama model (via Ollama) to optimize summarization accuracy and coherence.
  • Visualization Development: Developing more informative and user-friendly visualizations using LangGraph to represent document summaries effectively.
  • Initial System Testing: Conducting functionality and initial accuracy testing of the core components.

Progress Made

The core plan for the past three weeks has been successfully completed, with the integration of the backend with the LLM (Llama 3.1 8B via Ollama) achieved.

  • Enhanced Document Upload and Preprocessing: Robust document upload in FastAPI now supports Word, PDF, text, and image formats. Comprehensive preprocessing, including OCR for images, ensures accurate text extraction.
  • Advanced Summarization Integration: Seamless connectivity between preprocessed text and the Llama 3.1 8B model (via Ollama) has been established. Rigorous testing with various summarization techniques and parameters shows promising results.
  • Visualization Development: Informative visualizations using LangGraph have been developed and successfully integrated into the Streamlit user interface.
  • Initial System Testing: Functionality testing across all components and initial accuracy testing of the summarization output have yielded satisfactory results.

Challenges Overcome

  • Accurate Text Extraction from Diverse Document Formats: Complexity in handling varied PDF structures and images was overcome through careful implementation and testing of preprocessing techniques and the OCR engine.
  • Achieving High Accuracy and Coherence in LLM-based Summarization: Optimizing the summarization process with Llama 3.1 8B required experimentation with prompts and model parameters, leading to good initial results.
  • Developing Effective Visualizations for Document Summaries: Designing intuitive visualizations with LangGraph required careful consideration of user experience and the summarized content, resulting in initial visualizations that effectively convey the essence of summaries.

Tech Stack

  • FastAPI: Used for the backend to handle document upload and preprocessing.
  • Ollama: Used to host and serve the Llama 3.1 8B model.
  • Llama 3.1 8B Model: The Large Language Model used for document summarization.
